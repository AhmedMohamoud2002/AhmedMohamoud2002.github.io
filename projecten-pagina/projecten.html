Hoger-Lager

Dit is mijn eerste project, "Hoger-Lager". Hoger-Lager is een spel waarbij jij en de tegenstander dobbelstenen moet gooien vervolgens moet raden je hoger of lager getal hebt dan jouw tegenstander. Ik heb in dit project html/css & javascript gebruikt. Tijdens dit project heb ik geleerd hoe je een addEventlistener kunt toepassen om zo mijn buttons interactief te maken. Ik heb ook geleerd hoe je math.random kunt gebruiken om zo een random getal te generen.
